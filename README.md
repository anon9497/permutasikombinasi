<br>Permutasi :
<br>P (n,r) = n! / (n-r)!
<br>Keterangan:
<br>P = permutasi
<br>n = total objek keseluruhan
<br>r = total objek yang dijadikan urutan
<br>! = faktorial

<br>Kombinasi:
<br>nCr = n! / (r! (n-r)!)
<br>Keterangan:
<br>C = kombinasi
<br>n = total objek keseluruhan
<br>r = total objek yang diamati
<br>! = faktorial
